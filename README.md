# Diabetes Dashboard (Power BI)

## 📊 Project Overview
This project analyzes diabetes data from 98,000 patients to uncover trends and risk factors.

## 🔍 Key Insights
- 60% diabetes prevalence
- Higher HbA1c in older adults
- BMI strongly linked to insulin levels
- Family history increases risk

## 🛠 Tools Used
- Power BI
- Excel
